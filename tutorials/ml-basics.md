# ML Basics

This is where the journey starts. Each tutorial in this section teaches one core machine learning algorithm, from scratch, in plain English. No maths degree required. No coding experience needed.

You will learn how each algorithm works, see it explained with real-world examples, and run working code that you can experiment with straight away.

Brand new to this? Set up your coding environment first: [Get Started guide](setup). It only takes 5 minutes.

Then start with [What is Machine Learning?](what-is-ml) and work through the tutorials in order. Each one builds on the previous.
